# HDD LED App — Программный индикатор работы жесткого диска
Применение эффекта светодиодной индикации режимов чтения и записи данных с использованием вкладок интерфейса.
## Возможности
* Переключение между вкладками управления.
* Эмуляция работы жесткого диска (HDD LED):
    * Режим чтения (зеленое быстрое мигание).
    * Режим записи (красное быстрое мигание).
* Интерактивные графические кнопки с эффектом наведения мыши.
* Автоматический переход индикатора в режим ожидания (Idle) через 2 секунды бездействия.
## Установка зависимостей
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install pygame
```
## Запуск
```bash
python3 ThermalApp.py
```
## Использование
1. Перейдите на вкладку **Вкладка 1** для доступа к операции чтения.
2. Нажмите кнопку **«Чтение данных»** — индикатор начнет быстро мигать зеленым цветом.
3. Перейдите на вкладку **Вкладка 2** и нажмите кнопку **«Запись данных»** — индикатор начнет быстро мигать красным цветом.
4. Прекратите нажатия — через **2 секунды** лампочка автоматически погаснет (станет серой), показывая режим ожидания.
